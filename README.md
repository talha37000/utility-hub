# Utility Hub

A collection of HTML/CSS/JS mini-projects bundled in one repository.

## Included Projects

- Age Calculator
- Analog Clock
- Anime Pics Generator
- Auto Text Generator
- Background Image Scroll Effect
- Basic Calculator
- BMI Calculator
- Button Ripple Effect
- Currency Converter
- Game
- Image Slider
- New Year Countdown
- Pop-up Container
- Random Password Generator
- Random Image Generator
- Testimonial Slider
- To Do List
- Toggle Switch
- Video Background
- Weight Converter

## Assets

- `media/` contains images and icons used across the projects.

## How to Use

1. Clone the repository:
   ```bash
   git clone https://github.com/talha37000/utility-hub.git
   ```
2. Open any project folder and run the `index.html` file in your browser.

## Notes

- Each project is self-contained with its own HTML, CSS, and JS files.
- Some projects include a `README.md` inside their folder with more details.
